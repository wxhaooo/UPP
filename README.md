# UPP

## Introduction

在这个工程中，我打算实现或者解析UE4/Unity中C++缺失的一些重要机制。 __这个工程不涉及任何渲染、物理之类具体的模块拆解。主要拆解基本的引擎机制。__

预计有：

* __Delegate__
* __Reflection__
* __Module__
* __Coroutines__
> UE中不使用但仍然值得解析
* __Serialization__
